## 所使用基础库

基础样式: [water.css](https://github.com/kognise/water.css)  
代码高亮: [highlightjs](https://github.com/highlightjs/highlight.js/)

使用 highlightjs, 需要把配置里的 `highlight` 关掉

```yml
highlight:
  enable: false
  line_number: false
  auto_detect: false
  ? tab_replace
```
